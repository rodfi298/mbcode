Pricing

Start / Stop
from 45 €
Full disable  —  55 € The Start/Stop system is permanently disabled at software level. The engine never shuts off at traffic lights, in traffic or during short stops. No need to press the button on every start. The setting survives vehicle restarts.
Reverse logic  —  45 € Start/Stop is off by default. If needed, it can be manually activated with the button. This is the opposite of the factory setting where the system activates automatically every time and must be turned off manually each drive.
Memory mode  —  45 € The car remembers the last selected Start/Stop state. If the system was deactivated before switching off, it remains deactivated on the next start. The driver no longer needs to think about pressing the button every time.
AMG cluster menu
80 €
Activates the full AMG sport menu in the instrument cluster with additional data screens.
